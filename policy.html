StreamCheckBot – Privacy Policy

Effective Date: 26 January 2026

1. Introduction

StreamCheckBot (“we”, “our”, “the Bot”) respects your privacy and is committed to protecting any information you share while using the bot. By using StreamCheckBot, you agree to the practices described in this Privacy Policy.

2. Information We Collect

  • User input: When you use commands like /check, we receive the server address, username, and password you enter.
  • Technical info: We may temporarily process data such as Telegram user ID and timestamp to provide the service.

Important: We do NOT store passwords or IPTV credentials permanently. They are used only temporarily to fetch Xtream API info and then discarded.

3. How We Use Your Information

  • To fetch and return Xtream API data for your request.
  • To format responses and provide inline button functionality.
  • For logging errors and debugging (non-identifiable).

4. Data Sharing

We do not sell, rent, or share your personal data with third parties. Data is only processed within the bot to provide the service you requested.

5. Security

We implement reasonable measures to protect data in transit while fetching API responses. Telegram handles all communication encryption between you and the bot.

6. User Rights

You can stop using the bot at any time. Since we do not store personal credentials permanently, there is no permanent record of your sensitive data.

7. Updates to Privacy Policy

We may update this Privacy Policy from time to time. The latest version will always be available here or within the bot description.
